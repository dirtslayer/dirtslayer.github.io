Last year at this time I was aching to have time to work on computer projects but I was entrenched in day to day life. While I was working I delved into the world of *nix on Android. Now I have a server in my pocket and I'm sshd into my phone a lot of the time.

In January I tackled a project entirely done from my phone on breaks at work. [hazardtool.ca] is a hazard assessment tool. I am reasonably happy with it, there is still some work around tracking completed assesments, and making the printouts fit page more easily, but a person could easily use this tool for high quality hazard assessments (on your phone of course).

So the next sidetrack was curses / blessed and a bit of js, nothing too deep. Took half a javascript course and had a good time using Draw.io to create the diagram for the coding project here: [https://github.com/dirtslayer/js-stopwatch]

I am pushing very hard into the mobile space with my learning and wanted ways to make graphical programs so I learned an amazing game platform name Godot. My first project  game in Godot is called Batman Slider - a true to life recreation of a slider puzzle I enjoyed as a child. [https://github.com/dirtslayer/Batman-Slider].

With all this time on my hands at home with my kids lately, you'd think I didn't rack up thousands of kills ...

So the next little project was a data scrapper which cached web pages and plotted stuff on a graph. That was cool, I really liked the JSDom library. I also used some cool google infrastructure for that and got to dig around all the latest management consoles. 

Finally my last little project of the year, which, is the ultimate toilet game, is fruitmachine. This little discord bot has provided me with a moderate amount of joy. It started off as a App Engine Project but quickly discovered that discord bots are very active on the network so the fees where going to add up - so moved it onto sqlite3 and run it on my phone. [https://github.com/dirtslayer/fruitmachine]
